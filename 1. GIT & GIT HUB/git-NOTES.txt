-----------

Note:- jab v hum git status ko use krte hai to generally 4 types ke status show hoti hai:
                                                                       1. untracked  : new files that git doesn't yet trak.
                                                                       2. modified   : changed
                                                                       3. staged     : file is ready to be commited.
                                                                       4. unmodified : unchanged.

TEMPORARY COMMITS
#####################
Temporarily store modified, tracked files in order to change branches

GIT STASH :

-> git stash push a.txt                 # only for 1 file.
-> git stash push a.txt b.txt           # for multiple files.
-> git stash                            # for all files unstaged.
-> git stash pop                        # applies and deletes for all.
-> git stash list                       # see all listed stash files.
-> git stash apply                      # restored without removing.
-> git stash drop                       # discard the changes from top of stash stack

Q15. What is the difference between a commit, branch, tag, and HEAD in Git ? 
üîπ Commit
------------
- A snapshot of your project at a point in time.
- Think of it like saving a file version with a unique ID (SHA hash).
- Example: "This commit saved the code after fixing a bug."
üîπ Branch
------------
- A pointer (or label) to a series of commits.
- The default branch is usually main or master.
- Used to develop features separately without disturbing the main code.
- Example: feature/login branch for login functionality.
üîπ Tag
--------
- A pointer to a specific commit (usually a release point).
- Unlike branches, tags don‚Äôt move ‚Äî they‚Äôre fixed.
- Commonly used for marking versions like v1.0, v2.0.
üîπ HEAD
----------
- A special pointer that tells where you currently are in the repo.
- Usually, it points to the latest commit of the current branch.
- Example: If you‚Äôre on branch main, HEAD points to the tip of main.
